## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ItzPongo/Project-G.git
   ```
2. Open the project in Godot Game Engine
   ```
   Current version: v4.5.1.stable.mono.official
   ```
3. Run the game (F5)
